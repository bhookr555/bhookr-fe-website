# BHOOKR - Fresh Meal Subscription Platform

[![Production Ready](https://img.shields.io/badge/Production-Ready-brightgreen.svg)]()
[![Security](https://img.shields.io/badge/Security-Grade%20A-success.svg)]()

A **production-grade** food subscription web application built with Next.js 16, TypeScript, and industry-best practices.

## 🚀 Features

### User Features

- **Browse Meals**: Explore curated meal plans and dishes
- **Flexible Subscriptions**: Daily, weekly, or monthly plans
- **Secure Authentication**: Email/password and Google OAuth
- **Protected Dashboard**: Manage subscriptions and deliveries
- **Secure Checkout**: Multiple payment methods support
- **Order Tracking**: View billing history and upcoming deliveries
- **Dark Mode**: Full dark mode support
- **Responsive Design**: Mobile-first, works on all devices

### Technical Features

- **Next.js 14+ App Router**: Modern React framework with server components
- **TypeScript**: Full type safety throughout the application
- **Tailwind CSS**: Utility-first CSS framework
- **shadcn/ui**: Beautiful, accessible component library
- **NextAuth.js**: Complete authentication solution
- **Zustand**: Lightweight state management
- **React Hook Form + Zod**: Form validation and error handling
- **React Query**: Server state management and caching
- **Middleware**: Route protection and authentication checks
- **SEO Optimized**: Metadata, OpenGraph, and Twitter cards
